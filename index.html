Save data

This page has no server — everything lives in this browser's storage. To play with friends on separate machines, export your save after each session and share the file (Discord, email, whatever), then have the next person import it before their turn.

Built as a static page — host it on GitHub Pages and it just works. No backend, no accounts, no data leaves your browser unless you export it.
